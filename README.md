# MusicLib
个人曲谱收集与阅览工具，支持建立个人曲谱库、曲谱阅览与自动滚动播放，vibe-coding做的，欢迎多多交流需求与ai技术

# 使用步骤：
将本工程下载到本地
1. 双击MusicLib.exe，启动工具
2. 打开浏览器，输入localhost:5125，即可进入工具页面

默认曲谱存在/_internal/data文件夹下

# 其他说明：
1. 可以修改musiclib.ini更改配置，port更改端口号，library_dir指定曲谱保存位置
